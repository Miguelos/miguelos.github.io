Jekyll Bootstrap sticky footer

How to add a sticky footer

To use the sticky footer you’ll need to modify the default theme template and add some css.

include/theme/THEMENAME/default.html:

...
    <body>
      <div class="wrap">
        <!-- navbar and content here! -->
        <div class="footer-push"></div>
      </div>
      
      <footer>
        <div class="container">
          <!-- footer here! -->
        </div>
      </footer>
    </body>
    ...

Sass code:

/* Sticky footer */
    $footerHeight: 75px;

    //Page full height
    html, body {
        height: 100%;
    }
    
    #page-wrap {
        min-height: 100%;
        height: auto !important;
        height: 100%;
        // Negative indent footer by it's height
        margin-bottom: -($footerHeight);
        margin-left: 0;
        margin-right: 0;
        margin-top: 0;
    }
    
    //footer fixed height
    .footer-push,
    footer {
        height: $footerHeight;
        max-height: $footerHeight;
        overflow: hidden;
    }
